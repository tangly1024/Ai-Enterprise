# 产品方案

## 定位

为传统中小企业提供“重复工作替代器”：不是聊天机器人，而是能读企业资料、理解流程、生成草稿、执行系统操作、留下审计记录的自动化工具包。

## 用户

- 老板/负责人：想降本、减少等待、减少人为差错。
- 部门主管：想把 SOP 固化，减少培训和检查成本。
- 一线员工：想少复制粘贴、少查资料、少填重复表格。
- 交付顾问：想快速诊断流程、复制方案、持续运维。

## 高频场景

| 部门 | 重复操作 | AI/RPA 替代方式 |
| --- | --- | --- |
| 客服 | 查政策、回 FAQ、登记工单 | RAG 回答 + 工单草稿 |
| 销售 | 写报价、整理客户跟进、生成邮件 | 表格抽取 + 模板生成 |
| 财务 | 发票识别、报销初审、对账 | OCR/结构化抽取 + 审批流 |
| 人事 | 简历初筛、制度问答、入职清单 | RAG + checklist |
| 采购 | 询价比价、供应商资料归档 | 文档抽取 + 比价表 |
| 运营 | 日报周报、数据搬运、后台录入 | 定时任务 + RPA |

## 核心模块

1. 流程诊断：记录人工步骤、输入输出、系统账号、风险点。
2. 知识库：导入制度、SOP、产品资料、合同模板、历史 FAQ。
3. 任务收件箱：邮件、表单、聊天记录、上传文件进入统一队列。
4. Agent 编排：判断意图，检索知识，生成计划，调用工具。
5. 执行器：API、工作流、RPA、人工审批。
6. 审计台账：记录输入、检索来源、模型输出、执行动作、审批人。

## 不做

- 不做通用 AGI 办公平台。
- 不做自研 RPA 引擎。
- 不做没有审批的财务付款、合同签署、客户资料删除。
- 不在 MVP 支持所有行业。
