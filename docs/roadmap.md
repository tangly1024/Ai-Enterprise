# 落地路线

## 第 0 阶段：选 1 个客户或 1 个模拟行业

交付物：

- 10 个重复流程清单
- 每个流程的人力耗时、频次、系统、风险、输入输出
- 选出 1 个低风险高频流程做 MVP

## 第 1 阶段：知识库问答

目标：员工能问制度、产品、SOP，回答带来源。

验收：

- 导入 20-100 份文档
- 20 个真实问题测试
- 错误回答可追溯到来源或检索失败

## 第 2 阶段：结构化处理

目标：把邮件、表单、Excel、PDF 转成结构化工作项。

验收：

- 字段抽取准确率可人工复核
- 生成待办、报价草稿、工单草稿或报销初审
- 人可以一键批准、退回、修改

## 第 3 阶段：半自动执行

目标：Agent 生成计划，人确认后执行 API 或 RPA。

验收：

- 每个动作都有日志
- 高风险动作必须审批
- 失败能重试或人工接管

## 第 4 阶段：复制行业方案

优先行业：

1. 本地服务企业：客服、报价、排班、回访。
2. 贸易/批发：报价、采购、库存、对账。
3. 财税/代理记账：票据、报销、对账、材料收集。
4. 教培/咨询：线索跟进、课程资料、合同和回访。

## 最近 30 天任务

1. 写 3 个流程诊断模板。
2. 做 1 个知识库问答 demo。
3. 做 1 个邮件/表格抽取 demo。
4. 做 1 个人工审批 + RPA runner 的假执行 demo。
5. 用真实或模拟客户资料跑一遍验收。
